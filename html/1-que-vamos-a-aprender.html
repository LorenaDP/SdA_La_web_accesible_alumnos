La Web Accesible: Reconstrucción Dinámica

1. ¿Qué vamos a aprender?

¿Qué vas a aprender?

En esta actividad vas a trabajar con el DOM (Document Object Model) para aprender a modificar páginas web de forma dinámica.

Al terminar, serás capaz de:

  • Acceder a elementos de una página web (texto, imágenes, botones…).
  • Modificar su contenido y sus estilos desde JavaScript.
  • Crear y eliminar elementos del HTML en tiempo real.
  • Reorganizar partes de una web usando el árbol de nodos.
  • Responder a acciones del usuario (clics, eventos, etc.).

¿Para qué sirve esto en el mundo real?

Esto es exactamente lo que hacen muchas aplicaciones web:

  • cambiar el contenido sin recargar la página
  • adaptar la interfaz según el usuario
  • mejorar la experiencia de uso

Además, en este caso lo aplicarás a un objetivo concreto:

👉 hacer una web más accesible para personas con dificultades visuales
