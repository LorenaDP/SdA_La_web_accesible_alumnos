La Web Accesible: Reconstrucción Dinámica

4. Diccionario de términos

Diccionario de términos

Consulta este diccionario siempre que tengas dudas durante la actividad.

addEventListener
Método que permite asociar una función a un evento sin modificar el HTML.

 
Atributo (Attribute)
Información adicional de un elemento HTML (por ejemplo id, class, href).

 
appendChild
Método que añade un nodo como hijo de otro nodo.

 
cloneNode
Método que permite clonar un nodo existente.

 
createElement
Método que permite crear un nuevo elemento HTML desde JavaScript.

 
DOM (Document Object Model)
Modelo que representa un documento HTML como una estructura en forma de árbol, permitiendo acceder y modificar sus elementos mediante JavaScript.

 
Elemento (Element Node)
Nodo que representa una etiqueta HTML (por ejemplo <p>, <div>, <h1>). Puede contener otros nodos.

 
Evento
Acción que ocurre en la página web (clic, teclado, carga…) y que puede ser detectada mediante JavaScript.

 
getElementById
Método que permite seleccionar un elemento del DOM a partir de su identificador único (id).

 
innerHTML
Propiedad que permite leer o modificar el contenido HTML de un elemento.

 
Nodo
Cada uno de los elementos que forman el DOM. Puede ser un elemento HTML, texto, atributo, etc.

 
Nodo de texto (Text Node)
Nodo que contiene el contenido textual dentro de un elemento HTML.

 
nodeName
Propiedad que devuelve el nombre del nodo.

 
nodeType
Propiedad que indica el tipo de nodo mediante un valor numérico.

 
nodeValue
Propiedad que devuelve el valor de un nodo (normalmente texto).

 
querySelector
Método que permite seleccionar el primer elemento que coincida con un selector CSS.

 
querySelectorAll
Método que permite seleccionar todos los elementos que coincidan con un selector CSS.

 
Relación padre/hijo/hermano
Estructura jerárquica del DOM:

  • Padre → nodo superior
  • Hijo → nodo contenido dentro de otro
  • Hermano → nodos al mismo nivel

removeChild
Método que elimina un nodo del DOM.

 
replaceChild
Método que sustituye un nodo por otro.

 
textContent
Propiedad que permite modificar solo el texto de un elemento, sin interpretar HTML.
