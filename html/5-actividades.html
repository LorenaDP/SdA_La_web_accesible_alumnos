La Web Accesible: Reconstrucción Dinámica

5. Actividades

Información

A continuación se recogen las actividades propuestas a lo largo de la situación de aprendizaje, organizadas de forma secuencial

Actividad 1: Comprensión inicial

(Realizada tras el visionado de los vídeos en el apartado de contenidos)

Responde a las cuestiones planteadas para comprobar tu comprensión del DOM y su funcionamiento.

Entrega: Súbelo al Aula Virtual

Actividad 2: Mini reto práctico

(Disponible en el apartado de contenidos)

Modifica el código propuesto para que, al pulsar un botón, cambie el contenido de un párrafo.

Realiza las siguientes tareas en una página HTML:

  • Cambia el texto de un elemento al hacer clic en un botón
  • Modifica el color de un elemento
  • Añade un nuevo elemento al DOM

Entrega: Sube el archivo HTML al Aula Virtual

Actividad 3: Actividad interactiva

Completa el rosco del DOM y el test propuesto para comprobar tu comprensión de los conceptos clave.

Entrega: No requiere entrega (autoevaluación)

Actividad 4: Producto final

Producto 1: Infografía interactiva
Crea una infografía que explique el funcionamiento del DOM:

  • Estructura en árbol
  • Tipos de nodos
  • Métodos principales
  • La infografía deberá incluir al menos un elemento interactivo (enlaces, botones, etc.)

Herramientas: Canva / Genially
Entrega: Sube el enlace en Aula Virtual

 
Producto 2: Explicación práctica
Elige UNA de las siguientes opciones:

  • Vídeo explicativo
  • Podcast
  • Breakout
  • Mini videojuego

Debe incluir:

  • Explicación del DOM
  • Ejemplo práctico

Entrega: Sube el enlace en Aula Virtual

Actividad 5: Aplicación real

Desarrolla una página web en la que:

  • Se modifique el contenido dinámicamente
  • Se utilicen eventos
  • Se añadan o eliminen elementos del DOM

Entrega: Sube el archivo HTML y una breve explicación al Aula Virtual
