Mini reto práctico

Modifica el siguiente código para que al pulsar un botón cambie el contenido de un párrafo:

 
<p id="texto">Texto original</p>
<button id="btn">Cambiar texto</button>

Solución al mini reto. NO MIRAR HASTA HABER INTENTADO!!!

Explicación

  • getElementById("btn") permite seleccionar el botón del HTML.
  • addEventListener("click", ...) detecta cuándo la persona usuaria hace clic en el botón.
  • Dentro de la función, se vuelve a seleccionar el párrafo con getElementById("texto").
  • textContent se utiliza para modificar el texto del elemento seleccionado.

Gracias a esto, conseguimos que el contenido del párrafo cambie dinámicamente sin recargar la página.

 
Errores típicos

  • ❌ No poner bien el id
    Ejemplo: "Texto" en lugar de "texto" (recuerda que distingue mayúsculas/minúsculas).
  • ❌ Olvidar el addEventListener
    Sin él, el botón no hace nada.
  • ❌ Intentar usar innerHTML o textContent directamente sin seleccionar el elemento

Ejemplo incorrecto:     texto.textContent = "Texto cambiado";
 
(si no has definido antes texto, dará error)
❌ Colocar el script antes del HTML
Si el script está arriba del todo, los elementos aún no existen y no se pueden seleccionar.
❌ Confundir innerHTML y textContenttextContent → solo texto
innerHTML → permite HTML (etiquetas)
 
“Si no funciona, revisa primero los ids y el orden del script, ya que son los errores más habituales.”

CÓDIGO:

<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>Mini reto DOM</title>
</head>
<body>

<p id="texto">Texto original</p>
<button id="btn">Cambiar texto</button>

<script>
// Seleccionamos el botón
var boton = document.getElementById("btn");

// Añadimos el evento click
boton.addEventListener("click", function() {
// Cambiamos el contenido del párrafo
document.getElementById("texto").textContent = "Texto cambiado";
});
</script>

</body>
</html>
