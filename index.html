La Web Accesible: Reconstrucción Dinámica

¿Te apuntas?

El 96% de las webs tienen problemas de accesibilidad y millones de personas no pueden usarlas correctamente (Qué Tan Accesible es la Web: Principales Estadísticas de Accesibilidad en Internet [ 2025 ])

En esta actividad no solo vas a programar: vas a aprender a modificar una web para que funcione de verdad para cualquier persona.

Porque no se trata solo de que una web funcione, sino de que funcione para todo el mundo.
